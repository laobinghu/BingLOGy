# 前言

距离距离中考已过去5天。现在，终于可以梳理一下自己的初中三年了。

# 开始

我不是绵阳本地人。

但是我这十五年几乎都是在绵阳度过的——幼儿园、小学、初中，甚至两个月后的高中，仍然要再绵阳度过。

老实说，我在老家呆的时间加起来甚至不到一年。

**Why？**

我不知道，因为一切都要听父母的——他们说，这里的教育质量高。

可是，南昌的教育难道不如一个三线城市吗？

我不知道，因为没有对比标准。

绵阳说是一个三线城市，但教育水平远远超过了省内大部分城市。（绵中和**成都**七中在有些场景中是可以叫板的）

那就暂且相信他们。

# 入学

当时还有点招（就是掐尖，即各个初中把小学里成绩最好的学生给“抢”走）的说法，我就是其中被七中点招的一个。

~~挺绕口的~~

这意味着什么？~~小升初考试基本不用管了（如果按片区分的话会被直接分到对面八中）~~

对于我这个外地生来说，初中不用愁了。

# 坎坷

因为一些原因，我从小人缘就不怎么好。因此，我的性格有些孤僻。

好在小学的最后两年，情况有所改善。但是，初中来了。

这意味着——小学打下的关系，全部作废。

初中三年的最初，我的人缘还是一如既往的差——和别人合不来。当时城北还没分出去，所以有很多城北小学出来的直接升到七中。

诺大的班级 只有3个人是从同一个小学出来的，更别说另一个人是从另一个校区出来的。

这就导致一个问题：他们的梗，我几乎不知道。（喜提2G网络）

后来，有女生开始造我的黄谣，很快，一传十，十传百，全班都知道了。

初中三年，我一直在想，怎么才能融入集体。很多人都说要保持自己的独立个性。但请问：您自己试过吗？

想要融入大家，只能顺应玩笑，成为玩笑的当事人（以至于后来，我的名字=冷笑话）

直到现在，我仍不知道如何才能融入集体。

# 学习

作为一名点招生，我的成绩自然不算差。至少A优线还是能稳上的。这意味着什么(๑•̌.•̑๑)ˀ̣ˀ̣

上**南山**肯定没问题（老一辈人常说：~~奶奶，我要读南山~~没有三板斧，休想上南山）

当时，我不认为我能读绵中，所以一直到初二，我的目标永远是南山。

然后初三上的期末，班上前三（包括我）被南山请去~~白吃白喝~~参观，顺便暗示我们不要报绵中。

然后，绵中就给家长打电话，邀请我周末去绵中参观（说是参观，其实是洗脑+蹭饭）有一说一，绵中的饭是真的好吃😅

在绵中的一顿狂轰滥炸下，我的第一志愿填的绵中（关网前南山说如果考上了可以给我免学费，但想想我又不缺那一千多块钱，最终还是报的绵中）

至于最后的结果，自然是顺利考上了绵中。

# 游园

6月初，学校举行了一次游园活动。主要就是挣奖票，用奖票换吃的。现在，我只记得那天玩的很开心，但是散场后的萧条，却更难忘。

初中三年，劳劳碌碌，最终，却以一张奖票草草收尾。

剪不断，理还乱，是离愁，别是一般滋味在心头。

# 结束

初中三年，12个春秋交替，1015个日夜，却在弹指一挥间草草收尾。

老实说，我的初中是不合格的，因为我并没有成长太多。

但是，从成绩上看，我的初中是成功的，因为我顺利地考上的绵阳教学水平最高的高中——四川省绵阳中学。

以前，我无比期待中考的到来——因为中考意味着没有作业的暑假。但是现在看来，我的内心却无比空虚。思来想去，还是写下了这篇文章，来为这不完美的初中，写下一段总结。
